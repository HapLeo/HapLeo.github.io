# JVM常用参数表

| 参数                    | 作用                                       | 示例 |
| ----------------------- | ------------------------------------------ | ---- |
| -Xmn128m                | 指定新生代内存为128m                       |      |
| -Xms512m                | minimum memory size，指定最小堆内存为512m  |      |
| -Xmx1024m               | maximum memory size，指定最大堆内存为1024m |      |
| -XX:+UseConcMarkSweepGC | 指定使用CMS垃圾收集器                      |      |
| -XX:+UseSerialGC        | 指定使用Serial收集器                       |      |
